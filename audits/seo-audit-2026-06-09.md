## Medium (1)

- **BreadcrumbList schema missing on 6 pages** — `blog.html`, `runway.html`, `tools.html`, `bandwidth.html`, `career-friction.html`, `ai-relevance.html` — Add a BreadcrumbList JSON-LD block to each. For `blog.html`: `[Home → Blog]`. For tool pages: `[Home → Free Tools → Tool Name]`. For `runway.html`: `[Home → Abundance Runway Calculator]`. BreadcrumbList is the highest-value structured data for search appearance on inner pages; Google renders it as a breadcrumb trail below the title in SERPs, which increases click-through and helps Google understand site hierarchy.

## Low (4)

- **External links on `blog.html` missing `noreferrer`** — Lines 112, 124, 136, 148, 160, 221, 258, 270+ (all vj9.org outbound links) — Each uses `rel="noopener"` only. Add `noreferrer` to all: `rel="noopener noreferrer"`. The `noopener` already prevents the security issue; `noreferrer` additionally prevents the Referer header leaking to the destination. The index.html links already use the full form correctly — this is just an inconsistency on the blog page.

- **Article schema `keywords` missing on 3 older posts** — `post-unheard.html`, `post-lonely.html`, `post-conversation.html` — The three newer posts (post-ai.html, post-ai-loneliness.html, post-new-city-loneliness.html) all include a `keywords` array in their Article JSON-LD. The three older posts don't. Add a `keywords` array to each for consistency. Example for post-unheard: `"keywords": "decision-making clarity, feeling stuck career, coaching for clarity, ICF PCC coach Mumbai, professional coaching India"`.

- **`article:author` Open Graph tag on `post-unheard.html` uses org name, not person** — `post-unheard.html:21` — Has `<meta property="article:author" content="Clearhead" />` while other posts omit this tag entirely. Either remove it (to match other posts) or change the value to `Vaibhav Jain` and add it consistently across all posts. Using the publisher name as author is technically incorrect for an article by an individual.

- **Sitemap `lastmod` for homepage is one day stale** — `sitemap.xml:6` — Homepage lastmod is `2026-05-30` but `post-new-city-loneliness.html` (published 2026-05-31) is featured prominently on the homepage. Update homepage lastmod to `2026-06-09` (today) to signal the most recent crawl-worthiness.
